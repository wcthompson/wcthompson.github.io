Hey CSCI TA! This is my concept for a funner-than-average portfolio/resume: The bouncy resume! It provides a new way to display your accomplishments in a way that doesn't come across too braggy. Any number of items can be added, each one getting a random vertical and horizontal bounce speed on page load. To use this applet, you need to include the latest version of jQuery and link bounce.css and bounce.js (found in the source code below).

Each item simply needs wrapped by a div with id 'stage' and have the following form: 
  <div class="traveler" > 
        <div id="item-1" class="bouncer"> ITEM 1</div> 
  </div>

  <div id='content'> <div id="item-1"> Item 1 </div> </div>
where the 'bouncer' div contains whatever image or text you want bouncing around and the div with id matching the id of the bouncer contains whatever text or html you want displayed the bouncer is clicked. For an example in use, check out my personal site over here. The applet is purely css3 animation with a bit of jquery thrown in to handle the clicking.
